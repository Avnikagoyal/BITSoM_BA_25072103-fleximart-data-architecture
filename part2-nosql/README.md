# FlexiMart NoSQL (MongoDB) Analysis & Implementation

## Overview

This task evaluates MongoDB as a NoSQL solution for managing a highly diverse product catalog. It explains why MongoDB is suitable and demonstrates basic MongoDB operations.

## Files Included

- **nosql_analysis.md**  
  Justification report covering RDBMS limitations, MongoDB benefits, and trade-offs

- **mongodb_operations.js**  
  MongoDB script for loading data, querying products, review analysis, updates, and aggregations

- **products_catalog.json**  
  Sample product data used for MongoDB operations

## Key Concepts

- Flexible schema for diverse product attributes  
- Embedded documents for product reviews  
- Aggregation framework for analytics  

## How to Run

1. Start the MongoDB server  
2. Import data using MongoDB Shell or MongoDB Compass  
3. Run commands from `mongodb_operations.js`

## Tools Used

- MongoDB
